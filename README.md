# bloxd
bloxd, a Minecraft mod with modified graphics, is also a web game that adds many gameplay elements, all of which are free and enjoyable. It's online, and there are JavaScript commands available for programming, which will definitely bring you fun!   :D  :D  :D  :D

##  introduce 
Bloxd.io is a free-to-play, browser-based multiplayer sandbox game featuring voxel (Minecraft-style) graphics. It offers a wide variety of game modes, including:

- Survival & Creative: Build, explore, and survive in infinite worlds.
- Competitive Modes: Engage in popular mini-games like Bedwars, Sky Wars, and EvilTower (parkour).
- Social & Life Sim: Roleplay in Greenville or create custom private worlds.
